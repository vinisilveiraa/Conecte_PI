# Plataforma Conecte

Plataforma para conectar **clientes que necessitam de cuidados domiciliares** com **cuidadores e técnicos de enfermagem qualificados**.

O sistema permite busca geolocalizada de profissionais, envio de propostas, comunicação em tempo real e pagamento seguro.

---

# Funcionalidades Principais

- Cadastro de clientes e cuidadores
- Busca de cuidadores por proximidade
- Envio e aceitação de propostas
- Chat em tempo real
- Pagamento seguro
- Retenção de pagamento (escrow)
- Avaliação de profissionais

# Fluxo do Sistema

1 Cliente busca cuidadores próximos  
2 Cliente envia proposta  
3 Cuidador recebe notificação  
4 Cuidador aceita proposta  
5 Cliente realiza pagamento  
6 Chat é liberado  
7 Serviço é realizado  
8 Sistema libera pagamento após 24h  
9 Avaliação entre cliente e cuidador
